# TasklyQuest API 🎯

Backend en Ktor (Kotlin) para la app de tareas gamificadas TasklyQuest.

## 🛠️ Stack Tecnológico

- **Lenguaje:** Kotlin 2.0.21
- **Framework:** Ktor 2.3.0+
- **Base de Datos:** PostgreSQL 15+
- **Autenticación:** JWT
- **Serialización:** kotlinx.serialization
- **ORM:** Exposed
- **Testing:** Kotest, Mockk

## 🏗️ Arquitectura

Arquitectura modular escalable preparada para:
- Sistema de gamificación con eventos en tiempo real
- Microservicios futuros para características específicas
- Alta disponibilidad y escalado horizontal
